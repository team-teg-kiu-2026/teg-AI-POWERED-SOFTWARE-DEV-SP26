# Sample Audio Files

This directory should contain a sample audio file for Exercise 2.

## Provided Sample

**`sample_en_30s.mp3`** -- A 30-second clear English speech recording.

If this file is not present in the repository, you can create your own:

1. Open your phone's voice recorder
2. Record yourself reading the following paragraph clearly:

   > "Artificial intelligence is transforming how software is built.
   > In this course, we are learning to integrate AI capabilities into
   > real products. Text generation, vision, image creation, and now audio.
   > Each modality adds new possibilities and new responsibilities."

3. Save as MP3 or M4A
4. Transfer to your laptop
5. Place in this directory

Alternatively, use any clear English speech recording under 30 seconds.

## Requirements

- Duration: 15-30 seconds
- Language: English
- Quality: Clear speech, minimal background noise
- Format: MP3, WAV, or M4A
- Size: Under 5 MB
