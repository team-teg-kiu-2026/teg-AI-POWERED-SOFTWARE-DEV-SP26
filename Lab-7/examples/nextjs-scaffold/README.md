# Next.js Orchestration Scaffold

This folder contains starter modules for teams whose capstone uses Next.js.

## Files

- `lib/agent-state.ts` — shared state interface
- `lib/retry.ts` — timeout and retry helper
- `lib/guardrails.ts` — approval logic
- `lib/episode-log.ts` — event shape
- `app/api/ai/orchestrate/route.ts` — sample route
