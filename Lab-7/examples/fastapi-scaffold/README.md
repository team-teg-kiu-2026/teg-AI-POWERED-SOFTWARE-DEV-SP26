# FastAPI Orchestration Scaffold

This folder contains drop-in starter modules for teams whose capstone backend is in FastAPI.

## Files

- `schemas/agent_state.py` — shared state schema
- `services/retry_utils.py` — timeout and retry wrapper
- `services/guardrails.py` — approval logic
- `services/episode_logger_extension.py` — example event schema
- `agents/orchestrator.py` — minimal orchestration flow

These are not a full application.  
They are starter modules for Lab 7 integration.
